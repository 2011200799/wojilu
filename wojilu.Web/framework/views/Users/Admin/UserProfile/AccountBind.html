什么叫绑定帐号？
绑定帐号是指将自己在本站的帐号和其他网站关联起来（比如新浪微博、腾讯微博等大型SNS网站），方便在本站发布的信息能够及时的分享给其他网站的好友。
绑定帐号可以做什么？
放心，在绑定过程中，只有明确经过您的授权，帐号绑定才可以成功。绑定成功之后（比如新浪微博），您在本站发布的微博等信息，就可以同步到其他网站（比如新浪微博），再也不用您手动拷贝粘贴。如果您自己没有设定，也没有主动发布，本站不会向您的帐号推送任何广告或不良信息。
